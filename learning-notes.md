## File system

- The Master File Table (MFT) is a system file in the NTFS file system (having the name $MFT) that stores metadata information about all files and directories on an NTFS volume. The MFT acts as an index to all the files and directories on the volume, providing quick access to the information needed to retrieve a file

## Disable Windows Defender

- The most commonly used PowerShell cmdlet to disable Windows Defender is 'Set-MpPreference'

## Misc

- rdpclip.exe purpose: RDPClip is a program that enables "copy and paste" from a remote desktop session (RDP) that has been established on the system; it is tied to an active remote desktop connection and, other than event logs, identifying RDPCLIP.EXE is a key indicator of an RDP session used in intrusion cases and various internal investigations.
